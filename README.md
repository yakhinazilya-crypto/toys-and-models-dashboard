# toys-and-models-dashboard
Power BI dashboard - Toys and Models sales analysis

## À propos du projet
Tableau de bord interactif créé avec Power BI pour analyser 
les performances commerciales d'une entreprise spécialisée 
dans la vente de jouets et de modèles réduits.

## Contenu du tableau de bord
- Chiffre d'affaires par catégorie de produits
- Évolution des ventes dans le temps
- Indicateurs clés de performance (KPI)
- Analyse des produits les plus vendus
- Répartition géographique des ventes

## Outils utilisés
- Power BI Desktop
